Visio

The `Visio` object represents a video chat application that utilizes WebRTC technology for real-time audio and video communication. It provides methods for initializing the application, setting up a WebSocket connection with the server, handling different types of WebSocket messages, creating and managing a peer connection, and handling various video call actions such as creating an offer, accepting a call, declining a call, and ending a call.
